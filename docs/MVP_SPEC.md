# Coach FI MVP spec

## Cel MVP

Pokazac, ze Coach FI potrafi przeprowadzic osobe od chaosu finansowego do pierwszego mierzalnego postepu: ocena sytuacji, sciezka edukacji, lekcja, quiz, token reward i certyfikat NFT jako proof-of-progress.

## Persona

Uzytkownik jest doroslym, dzieckiem, osoba niedowidzaca lub osoba finansowo wykluczona. Chce zrozumiec pieniadze, inflacje, oszczedzanie, kredyt, stopy procentowe, fundusz awaryjny, emeryture i podstawy dlugoterminowego inwestowania. Ma niepewnosc, slabe nawyki, brak struktury albo za malo kapitalu awaryjnego.

## Zakres edukacyjny core MVP

Core MVP obejmuje:

- money mindset,
- saving habit,
- inflation and purchasing power,
- credit, interest and WIBOR,
- emergency fund,
- long-term investing basics,
- retirement and future security,
- family financial plan.

Core MVP nie uczy tradingu, leverage, futures ani spekulacji. Crypto moze pojawic sie pozniej jako modul edukacyjny, ale nie jest czescia core MVP.

## Core loop

1. Uzytkownik odpowiada na 6 pytan.
2. Aplikacja liczy readiness score.
3. Coach generuje sciezke nauki.
4. Uzytkownik wykonuje pierwsza mikro-lekcje.
5. Uzytkownik zdaje quiz.
6. Aplikacja przyznaje Coach FI Tokens i zwieksza streak/XP.
7. Po etapie uzytkownik moze mintowac certyfikat NFT na Solana jako proof-of-progress.

## Zakres obecnego demo

- Next.js frontend z osobnymi trasami,
- lokalny stan React i mock data,
- scoring i personalizacja na mock danych,
- symulowana rozmowa coacha,
- quiz z natychmiastowym wynikiem,
- mock Coach FI Tokens,
- mock NFT certificate mint flow na Solana Devnet,
- tryby Adult, Kids i Accessibility,
- high contrast, larger text i read aloud controls.

## Poza zakresem core MVP

- trading,
- leverage,
- futures,
- spekulacja,
- rekomendacje inwestycyjne,
- gwarancje zysku,
- prawdziwe doradztwo finansowe,
- prawdziwe transakcje blockchain,
- logowanie,
- platnosci,
- zewnetrzne modele AI,
- prawdziwe polaczenie z bankiem.

## Kryterium sukcesu demo

Demo jest udane, jesli w 2-3 minuty pokazuje:

- dlaczego produkt zaczyna od zdrowia finansowego, a nie od kupowania aktywow,
- jak wyglada personalizowana sciezka,
- jak lekcje, quizy, tokeny i streaki tworza retencje,
- gdzie naturalnie wchodzi NFT certificate/proof-of-progress,
- ze Coach FI buduje confidence, healthy habits i long-term awareness bez promowania ryzykownego inwestowania.
