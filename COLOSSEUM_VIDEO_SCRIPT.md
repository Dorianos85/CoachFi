## NAGRYWANIE — INSTRUKCJA KROK PO KROKU

### Opcja A — Loom (najszybciej, 30 min)
1. Zainstaluj [loom.com](https://loom.com) (darmowy)
2. Sceny 1, 2, 5, 6 → nagraj slajdy z Claude.ai jako tło
3. Sceny 3, 4 → screen recording aplikacji + mów skrypt
4. Loom zmontuje automatycznie
5. Wyślij link do Colosseum

### Opcja B — Canva / PowerPoint (bardziej dopracowane, 2h)
1. Wklej `PITCH_DECK.md` do Claude.ai → wygeneruj 6 slajdów
2. Eksportuj jako PNG/PDF
3. Nagraj prezentację z voice-over (OBS lub wbudowany recorder)
4. Zmontuj w CapCut / DaVinci Resolve (darmowe)

### Opcja C — Screen recording only (ultra prosto)
1. Otwórz aplikację CoachFI lokalnie
2. Nagraj screen w OBS lub Windows Game Bar (Win+G)
3. Mów skrypt podczas nagrania
4. Przytnij do 2 minut w CapCut
